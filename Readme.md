## Install

This generator requires [node](https://nodejs.org), [npm](https://npmjs.com).

You can install it by running:

```sh
npm i @armelwanes/magick-array
```

## Usage

### sum of array list

```js
import {sum} from '@armelgeek/magick-array';
.....
const total = sum([1, 2, 3]);
// 6
```
